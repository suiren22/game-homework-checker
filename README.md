# 모바일 게임 숙제 체크 PWA

안드로이드에서 앱처럼 사용할 수 있는 Progressive Web App (PWA)입니다.

## 주요 기능

### 🎮 게임 숙제 관리
- **니케** - 오전 5시 초기화
- **원신** - 오전 5시 초기화
- **젠레스 존 제로** - 오전 5시 초기화
- **엔드필드** - 오전 5시 초기화
- **명조** - 오전 5시 초기화
- **세븐나이츠R** - 오전 9시 초기화 ✅
- **블루아카이브** - 오전 4시 초기화
- **소녀전선2** - 오전 5시 초기화

### ✨ 개선된 기능
1. **PWA 지원** - 홈 화면에 앱으로 설치 가능
2. **오프라인 지원** - Service Worker를 통한 오프라인 사용
3. **진행률 표시** - 상단에 완료 진행률 바 표시
4. **긴급 알림** - 초기화 1시간 전 타이머 깜빡임
5. **진동 피드백** - 완료 시 진동 피드백 (지원 기기)
6. **전체 초기화** - 모든 숙제 한번에 초기화 버튼
7. **자동 초기화** - 각 게임별 초기화 시간에 자동으로 리셋
8. **이미지 파일 사용** - Base64 대신 이미지 파일로 빠른 로딩

## 안드로이드에서 앱으로 설치하는 방법

### Chrome 브라우저 사용 시
1. Chrome에서 이 페이지를 엽니다
2. 상단에 나타나는 "앱으로 설치하기" 배너의 **설치** 버튼을 클릭
3. 또는 Chrome 메뉴(⋮) → **앱 설치** 또는 **홈 화면에 추가** 선택
4. 홈 화면에 앱 아이콘이 추가됩니다

### Samsung Internet 브라우저 사용 시
1. Samsung Internet에서 이 페이지를 엽니다
2. 메뉴(≡) → **페이지 추가** → **홈 화면** 선택
3. 홈 화면에 앱 아이콘이 추가됩니다

## 사용 방법

1. **숙제 완료 체크**: 게임 카드를 탭하면 "CLEAR" 표시와 함께 완료 처리
2. **완료 취소**: 완료된 카드를 다시 탭하면 "안했어요" 선택으로 취소 가능
3. **타이머 확인**: 각 게임별 초기화까지 남은 시간 실시간 표시
4. **전체 초기화**: 하단의 "전체 초기화" 버튼으로 모든 숙제 리셋
